Assalamualaikum

🤖 Tabarag: Asisten AI Pajak Offline & Ringan
Tabarag adalah solusi chatbot berbasis kecerdasan buatan (Artificial Intelligence) yang dirancang khusus untuk membantu pengguna memahami berbagai informasi seputar pengetahuan umum perpajakan.

Berbeda dengan chatbot konvensional yang bergantung pada cloud dan koneksi internet, Tabarag beroperasi sepenuhnya secara lokal dan offline, serta dioptimalkan agar ramah terhadap perangkat dengan sumber daya terbatas.

✨ Fitur Utama
🧠 Spesialis Perpajakan: Dilatih untuk mengenali dan menjawab berbagai pertanyaan seputar konsep, istilah, dan pengetahuan umum perpajakan.

🔌 100% Offline-First: Dapat digunakan kapan saja dan di mana saja tanpa memerlukan koneksi internet, memastikan privasi data Anda tetap terjaga di perangkat lokal.

🪶 Super Ringan (Low Resource): Dirancang dengan penggunaan kapasitas RAM dan storage yang sangat minimal, sehingga tetap berjalan lancar pada perangkat berspesifikasi rendah.

💻 Eksekusi Lokal: Berjalan sepenuhnya di mesin Anda sendiri tanpa ketergantungan pada server eksternal atau biaya layanan pihak ketiga.

🚀 Mengapa Tabarag?
Tabarag hadir untuk menjembatani kebutuhan informasi perpajakan yang cepat dan akurat, tanpa harus khawatir tentang kuota internet, latensi jaringan, atau batasan perangkat keras yang mahal.
